# Calculatrice Web

Une calculatrice web développée en HTML, CSS et JavaScript.

## Fonctionnalités

- Addition
- Soustraction
- Multiplication
- Division
- Bouton DEL (suppression)
- Bouton C (réinitialisation)
- Interface responsive
- Affichage dynamique

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
